# TodoCode

## Code Annotations

| Nom    | Description                                                  |
|--------|--------------------------------------------------------------|
| TODO   | Indique les tâches qui doivent être complétées dans le futur |
| FIXME  | Marque du code qui doit être corrigé ou amélioré             |
| HACK   | Identifie une solution de contournement ou non optimale      |
| UNDONE | À définir                                                    |
